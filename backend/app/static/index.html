Copilot User Guide

Screen 1 Secure Officer Access
Investigators log in securely using credentials or quick biometrics.

💡 How to use:
• Standard Login: Click Access Portal using the default badge ID.
• Biometric Access: Tap the Fingerprint Scan Area. An electric green scanner bar overlay will sweep down to verify biometrics and redirect you to the main dashboard.
Screen 2 Investigation Dashboard
The control center provides an overview of active cases, predictive maps, and alerts.

💡 Features explained:
• AI Alert Ticker: Dynamically highlights high-priority contradictions discovered during testimony matches.
• New Case Button: Launches the evidence catalog scanner rooms.
• Active Cases List: Shows progress cards. Click the Mehta Jewellers card to load the detailed analysis.
• Crime Hotspot Map: Highlights high-risk zones compiled from surrounding crime logs.
Screen 3 New Case Upload Room
Upload media, documents, or logs to initiate AI parsing.

💡 How to use:
• Click any selector (PDF, Voice Complaint, or CCTV Evidence) to simulate picking a file.
• A blue card details panel will emerge showing file specifications.
• Click Analyze Case with AI to trigger the multi-step loading sequence, routing raw bytes to Gemini API processing.
Screen 4 AI Incident Analysis
Extracts suspects, victims, and prioritizes case focus.

💡 Option meanings:
• Severity Index: A score out of 100 representing crime severity (e.g. 68% grand larceny).
• Extracted Suspects: Names parsed by Gemini NLP algorithms.
• Missing Information: Points out structural gaps in files that officers need to address immediately.
Screen 5 Chronological Timeline
AI synthesizes witness statements, CCTV logs, and telecom records to construct a chronological order of events.

💡 Option meanings:
• Shows exact time cards connected by vertical timeline channels.
• Displays source tags (e.g., Witness, CCTV, Telecom) representing which document verified the event.
Screen 6 Evidence Link Map (USP)
A visual network graph mapping relationships between victims, phones, CCTVs, suspects, and vehicles.

💡 Option meanings:
• Green node: Victims.
• Red node: Suspects.
• Cyan / Yellow / Purple nodes: Phones, vehicles, and CCTV cameras.
• Edges: Dynamic lines connecting nodes, detailing relationship descriptions (e.g., Vikram owns Vehicle).
Screen 7 AI Suggestions & Checker
Provides next actions, contradiction checkers, and scene simulators.

💡 Features explained:
• Recommended Next Steps: Checkbox tasks with priority ratings.
• Contradiction Detector: Paste/read witness A and B accounts, and click Analyze to highlight conflicting statements.
• 3D Scene Reconstruction: Click the purple card play button to open the coordinate path mapping sheet representing entry paths.
Screen 8 AI Case Report & Print
Compiles case summaries, discrepancies, and timelines into a formatted document.

💡 How to use:
• Review compiled sections inside the report card.
• Click Export to PDF to trigger a system dialog simulating case compilation and local storage printing.
